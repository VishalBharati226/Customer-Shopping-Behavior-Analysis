# 🛍️ Customer Shopping Behavior Analysis | Data Analytics Project

## 📌 Project Summary
An end-to-end data analytics project focused on analyzing customer shopping behavior to uncover trends, optimize decision-making, and generate actionable business insights.

This project demonstrates strong skills in data cleaning, querying, analysis, and visualization using industry-standard tools.

## 📊 Key Business Questions Solved
- What are the purchasing patterns of customers?
- Which age group contributes the most to revenue?
- How do ratings impact purchase behavior?
- Which product categories drive the highest sales?
